Empire Group WA is an industrial construction project management company. I designed and built the site in WordPress using Elementor, working directly with the client throughout the project. The main challenge was creating a strong visual result from a very minimal black-and-white brand system and limited written content, which meant shaping the design and copy together as the site developed.
